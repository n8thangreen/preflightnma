# Generate Quarto Report for NMA Preflight Diagnostics

Generates a comprehensive Quarto HTML report containing all preflight
diagnostic outputs: network sparsity, connectivity, transitivity, and
covariate modifier checks.

## Usage

``` r
nma_preflight_report(
  preflight,
  output_file = "preflight_report.html",
  title = "NMA Preflight Diagnostics Report",
  author = NULL,
  browse = TRUE,
  ...
)
```

## Arguments

- preflight:

  An `nma_preflight` object generated by
  [`nma_preflight()`](https://n8thangreen.github.io/preflightnma/reference/nma_preflight.md),
  or an `nma_data` network object (in which case
  [`nma_preflight()`](https://n8thangreen.github.io/preflightnma/reference/nma_preflight.md)
  will be run first).

- output_file:

  Character; the path/filename of the output report (default:
  "preflight_report.html").

- title:

  Character; the title of the report (default: "NMA Preflight
  Diagnostics Report").

- author:

  Character; the author of the report.

- browse:

  Logical; if `TRUE` (default), opens the generated report in the
  default web browser.

- ...:

  Additional arguments passed to
  [`nma_preflight()`](https://n8thangreen.github.io/preflightnma/reference/nma_preflight.md)
  if a network object is provided.

## Value

The path to the generated report file (invisibly).
