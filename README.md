## Adding the badge image

The README references the badge image at `assets/preflightnma-badge.png`. To make the badge appear on GitHub, add the image file to that path in the repository.

To add the image locally:

- macOS / Linux:

  mkdir -p assets && cp /path/to/your/badge.png assets/preflightnma-badge.png

- Windows (PowerShell):

  New-Item -ItemType Directory -Path assets -Force; Copy-Item C:\path\to\badge.png assets/preflightnma-badge.png
